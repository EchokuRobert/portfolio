VitalTrack Health

Next-generation secure medical data management.

The Challenge

Patients and providers required a system that could store highly sensitive biometric data with zero-latency access while meeting strict international security standards.

The Solution

I developed a Flutter-based mobile suite utilizing decentralized storage and end-to-end encryption. The app allows for real-time syncing even in low-bandwidth environments.

Technical Stack
  • Frontend Flutter / Dart
  • Encryption AES-256 / Biometrics
  • Infrastructure AWS S3 / Node.js
  • Database Firebase / Firestore
